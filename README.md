# Reestar Website

A simple restaurant website built using **HTML** and **basic CSS**. This project is designed as a beginner-friendly frontend practice to improve web development skills.

## Features

* Clean and simple user interface
* Food menu section
* Contact section
* Responsive layout using basic CSS

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to practice the fundamentals of HTML and CSS, including page structure, styling, and layout design.
